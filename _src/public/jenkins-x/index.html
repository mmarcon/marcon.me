Jenkins-X

A OSX client for Jenkins

Jenkins-X is a small OSX client for Jenkins, the extendable, open source continuos integration server.

Jenkins-X displays the status of your Jenkins jobs and provides a simple meter that shows the stability of your builds. Additionally it integrates perfectly with OSX showing the current status of the build in the menu bar.

Jenkins-X is open source and built using open source components.

You can build Jenkins-X yourself by downloading the Titanium project from Github or simply download the prebuilt DMG for OSX.

Note that I am actively working on Jenkins-X, so there may be some little changes here and there happening on a daily basis. I will keep the repository on Github up to date as I make changes and clean up the code. I will release a new DMG image only when I make major changes or fixes.

Some additional information and implementation details on Jenkins-X are available on Side Project.

Known Issues

  • There is a big memory leak that makes the application crash after it has been running for a few hours. Just restart Jenkins-X after half a day or so :p
  • The OS-notifications steal focus from any other application. However if you keep your build always green the build status never changes and notifications are not going to be triggered :p
  • There is no documentation available, but the settings are so simple that it is probably not even required.
